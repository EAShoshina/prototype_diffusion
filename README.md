# Применение архитектуры трансформеров в диффузионных моделях для оценки хвостовых финансовых рисков
## Описание проекта

Проект реализует прототип модели оценки хвостовых финансовых рисков на основе диффузионных генеративных моделей с использованием архитектуры трансформеров.

Работа выполняется в рамках научно-исследовательской работы (НИР) по теме:
**«Применение архитектуры трансформеров в диффузионных моделях для оценки хвостовых финансовых рисков»**.

Актуальность задачи обусловлена тем, что распределения финансовых доходностей обладают тяжёлыми хвостами, что приводит к систематическому занижению риска в классических моделях.

---

## Цель исследования

Разработка и исследование метода оценки хвостовых финансовых рисков на основе диффузионных генеративных моделей с использованием архитектуры трансформеров, обеспечивающего более точное моделирование распределения доходностей в хвостовых областях.

---

## Задачи проекта

В рамках работы решаются следующие задачи:

* анализ существующих методов оценки хвостовых рисков (VaR, Expected Shortfall);
* изучение диффузионных моделей и трансформеров для временных рядов;
* разработка генеративной модели финансовых данных;
* программная реализация модели;
* проведение вычислительных экспериментов;
* оценка качества модели на задаче моделирования хвостов распределения.

---

## Реализованный функционал

В текущей версии прототипа реализовано:

* диффузионная модель генерации данных (DDPM);
* трансформерный энкодер для учёта временных зависимостей;
* прямой и обратный диффузионные процессы;
* генерация сценариев доходностей методом Монте-Карло;
* расчёт метрик риска:

  * Value at Risk (VaR)
  * Expected Shortfall (ES);
* реализация статистического теста:

  * Kupiec test (проверка корректности VaR).

---

## Архитектура модели

Модель включает следующие компоненты:

1. **Transformer Encoder**
   Обрабатывает историческое окно доходностей и извлекает временные зависимости.

2. **Diffusion Model (DDPM)**
   Обучается восстанавливать зашумлённые данные, моделируя распределение доходностей.

3. **Noise Scheduler (cosine schedule)**
   Управляет процессом добавления шума.

4. **Reverse Diffusion (sampling)**
   Генерация новых сценариев через стохастический процесс.

## Научная новизна

В отличие от классических методов (GARCH, историческое моделирование):

* отсутствуют предположения о форме распределения;
* учитываются временные зависимости через трансформер;
* обеспечивается более полное покрытие распределения, включая хвостовые области.

Комбинация диффузионных моделей и архитектуры трансформеров является современным направлением исследований в области моделирования временных рядов.
