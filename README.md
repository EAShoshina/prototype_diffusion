# Применение архитектуры трансформеров в диффузионных моделях для оценки хвостовых финансовых рисков

## Описание проекта

Данный проект представляет собой прототип системы оценки хвостовых финансовых рисков на основе диффузионных генеративных моделей с использованием архитектуры трансформеров.

Проект выполнен в рамках научно-исследовательской работы по теме:
**«Применение архитектуры трансформеров в диффузионных моделях для оценки хвостовых финансовых рисков»**.

Основная идея заключается в моделировании распределения доходностей финансовых активов без предположений о его параметрической форме. Это позволяет более точно учитывать экстремальные значения (тяжёлые хвосты), что критически важно для оценки рыночного риска.


## Цель проекта

Разработка прототипа модели для генерации распределения доходностей финансовых активов и оценки хвостовых рисков (Value at Risk и Expected Shortfall) с использованием диффузионных моделей и трансформеров.


## Реализованный функционал

В рамках проекта реализованы следующие компоненты:

* загрузка и обработка финансовых данных (Московская биржа);
* формирование обучающей выборки на основе временных окон;
* диффузионная модель (DDPM) для генерации данных;
* трансформерный энкодер для учёта временных зависимостей;
* генерация сценариев доходностей методом Монте-Карло;
* расчёт метрик риска:

  * Value at Risk (VaR)
  * Expected Shortfall (ES);
* проведение бэктестинга:

  * Kupiec test.


## Архитектура системы

Система состоит из следующих модулей:

1. **Data Processing Module**
   Подготовка временных рядов доходностей.

2. **Transformer Encoder**
   Извлечение временных зависимостей из исторического окна.

3. **Diffusion Model (DDPM)**
   Обучение генеративной модели через процесс добавления и удаления шума.

4. **Scenario Generation Module**
   Генерация будущих сценариев доходностей.

5. **Risk Evaluation Module**
   Расчёт VaR, Expected Shortfall и проведение статистических тестов.

Разработанный прототип может быть использован для:

* моделирования экстремальных рыночных сценариев;
* анализа хвостовых распределений.


## Научная новизна

В отличие от классических методов (например, GARCH или историческое моделирование), предложенный подход:

* не требует предположений о форме распределения доходностей;
* учитывает временные зависимости с помощью трансформеров;
* обеспечивает более точное моделирование хвостовых областей распределения.


## Ограничения

* текущая версия прототипа требует дальнейшей настройки гиперпараметров;
* не реализована полная оптимизация архитектуры модели;
* необходим расширенный бэктестинг на различных рыночных условиях.
