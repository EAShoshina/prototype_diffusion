# Применение архитектуры трансформеров в диффузионных моделях для оценки хвостовых финансовых рисков

## Описание проекта

Данный проект реализует прототип генеративной модели для оценки хвостовых финансовых рисков на основе диффузионных моделей и архитектуры трансформеров.

Основная идея заключается в моделировании распределения доходностей финансовых активов без предположений о его параметрической форме. В отличие от классических подходов (GARCH, историческое моделирование), диффузионные модели позволяют учитывать сложную структуру данных, включая тяжёлые хвосты и временные зависимости.

В основе реализации лежит подход Denoising Diffusion Probabilistic Models, адаптированный для финансовых временных рядов с использованием трансформерного энкодера.

---

## Цель проекта

Разработка прототипа системы для генерации финансовых временных рядов и оценки хвостовых рисков (VaR, Expected Shortfall) на основе диффузионных моделей.

---

## Основной функционал

В текущей версии реализовано:

* генерация распределения доходностей финансового актива;
* моделирование временных зависимостей с помощью трансформера;
* реализация прямого и обратного диффузионного процесса;
* генерация сценариев методом Монте-Карло;
* базовое обучение модели на данных.

---

## Архитектура модели

Модель состоит из следующих компонентов:

1. **Context Encoder (Transformer)**
   Обрабатывает историческое окно доходностей и извлекает временные зависимости.

2. **Diffusion Model**
   Обучается восстанавливать зашумлённые данные (предсказание шума).

3. **Noise Scheduler**
   Управляет процессом добавления и удаления шума (cosine schedule).

4. **Sampling (DDPM)**
   Генерация новых сценариев через обратный диффузионный процесс.

После обучения модель генерирует распределение доходностей:

```
mean ≈ 0.01
std ≈ 1.05
```

Это указывает на корректную работу диффузионного процесса (отсутствие дивергенции).

## Новизна подхода

В отличие от традиционных методов:

* отсутствуют предположения о форме распределения;
* учитываются временные зависимости;
* моделируются экстремальные события.

Комбинация диффузионных моделей и трансформеров позволяет более точно аппроксимировать хвостовые области распределения.
